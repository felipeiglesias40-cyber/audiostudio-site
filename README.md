# AudioStudio Online

Site oficial do AudioStudio Online - Separação profissional de instrumentos com IA.

## 📋 Sobre o Projeto

AudioStudio Online é uma plataforma que utiliza inteligência artificial para separar instrumentos de músicas, permitindo extrair vocais, bateria, baixo, guitarra e muito mais.

## 🚀 Tecnologias Utilizadas

- **Frontend:** HTML, CSS, JavaScript
- **Hospedagem:** Vercel
- **Backend API:** Render
- **Banco de Dados:** MongoDB Atlas
